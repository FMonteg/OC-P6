Classifieur identifiant la race d'un chien à partir d'une photo :
-----------------------------------------------------------------

1 - Placer l'image à classer dans le même dossier que le code source sous le nom "image.jpg".

2 - Lancer le code Python "Code final.py". La race la plus probable s'affichera dans la console.




----------------
Notes diverses :
----------------

- Le fichier "classes.pi" et le dossier "keras_save" contiennent les modèles pour la classification et ne doivent pas être modifiés.

- L'image présente par défaut est un chien "papillon".
